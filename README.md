# CNN 
支持

    - ResNet50
    - Xception
    - InceptionV3
    - VGG16 

特征生成，和融合训练预测

# 代码
环境：python2 keras2(tensorflow backend)
## 结构
name | usage
---|---
Preprocessing train dataset gap.py | 构造数据集工具,需要根据不同情况修改
gap.py | 生成特征并保存
gap_train.py | 读取生成的特征训练,保存模型
gap_predict.py| 读取模型和gap.py生成的特征预测
